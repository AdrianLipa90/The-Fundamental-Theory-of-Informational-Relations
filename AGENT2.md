# AGENT2.md

## Role
AGENT2 is responsible for cross-reference integrity in this repository.

## Scope at repository root
AGENT2 maintains the navigation and relation topology connecting:
- root indexes,
- local sector indexes,
- registries,
- interfaces,
- derivation entry points,
- whitepaper entry points,
- axiom-level entry points.

## Core duties
1. prevent orphan files,
2. prevent parallel navigation spines,
3. ensure every imported object has a registry path,
4. ensure every exportable object has explanatory and status references,
5. ensure cross-links remain synchronized after edits.

## Non-scope
AGENT2 does not claim authorship over physical derivations, constants, or runtime implementation unless the task is explicitly cross-reference related.

## Working rule
Every AGENT2 change must improve or preserve:
- link clarity,
- dependency direction,
- single-source-of-truth discipline,
- sector-local and global index coherence.

## Primary files watched
- `INDEX.md`
- `index.md`
- `STRUCTURE.md`
- `registries/*.yaml`
- `interfaces/*.yaml`
- local `index.md` files
