# DEF-0007 Identity

Status: defined

Symbol: Id

## Role
Identity is the relatively stable pattern-preserving continuity by which a system or object remains re-identifiable across admissible transformations.

## Formal definition
Identity is the persistence of a bounded relational-state pattern under admissible reconfiguration, such that the object or regime remains distinguishable as the same structured entity through time or transformation.
