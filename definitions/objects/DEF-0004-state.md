# DEF-0004 State

Status: defined

Symbol: ψ

## Role
State is the canonical representation of the currently admissible configuration of a system under the declared relational frame.

## Formal definition
State is the minimal structured object sufficient to encode the presently admissible configuration variables, constraints, and distinguishable relations required for downstream evolution or comparison.
