# Definitions Index

Definitions stabilize canonical object naming, symbols, roles, and upstream bindings before full derivation closure.

## Role in local topology
- upstream of derivations and implementation
- downstream of axioms and explicit assumptions
- synchronized with layer registry entries in `../registries/definitions_layer_registry.yaml`

## Current status
Definition-level source files are only partially normalized.
This index is the canonical local entrypoint for future definition objects and sectors.
