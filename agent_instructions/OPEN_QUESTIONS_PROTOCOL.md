# Open Questions Protocol

## Purpose
Track epistemic gaps explicitly.

## Categories

1. OPEN QUESTION
   - unresolved
   - linked to specific derivation or definition

2. PARTIALLY ANSWERED
   - candidate derivation exists
   - not yet validated

3. CLOSED
   - derivation exists
   - test or simulation supports result

## Required fields
- ID (Q-xxxx)
- linked definitions
- linked derivations
- current status
- notes

## Rule
Untracked uncertainty is considered an error.
