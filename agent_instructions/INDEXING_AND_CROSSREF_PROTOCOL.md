# Indexing and Cross-Reference Protocol

## Purpose
Ensure global coherence of the repository as a relational-topological system.

## Requirements

### 1. Every object must have an ID
- DEF-xxxx
- D-xxxx
- T-xxxx
- NB-xxxx

### 2. Every artifact must reference:
- upstream dependencies
- downstream usages

### 3. Cross-reference graph
Each artifact must declare:
- definitions used
- derivations used
- code modules used
- simulations linked

### 4. Glossary integration
New objects must be registered in glossary with:
- symbol
- meaning
- relations
- constraints

### 5. Artifact indexing
All artifacts must be indexable via:
- ID
- type
- dependency
- sector

## Rule
If an artifact cannot be located via ID and dependency path, it is not considered part of the theory.
