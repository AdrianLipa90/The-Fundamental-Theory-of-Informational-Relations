This module repays the debt opened after Debt 7: the list of candidate axes must be derived from source geometry, not imported from the finished Standard Model.

Main result:

- primitive source axis is first named `SOURCE_DERIVED_CHIRAL_CP1_AXIS`,
- weak isospin is only the Standard Model interpretation of that abstract axis,
- Berry, zeta, twin-prime/Collatz, Ramanujan, hypercharge, and tetrahedral/color layers are assigned distinct roles and excluded as primitive CP1 pole selectors.
