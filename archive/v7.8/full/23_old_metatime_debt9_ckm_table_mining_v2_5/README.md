# v2.5 — Old Metatime Documents Mining for Debt 9 / CKM / Representation Table

This module is an audit/mining pass over the old `Metatime-main` documents already stored in `00_original_metatime`.

It does not promote old claims to canon. It classifies old material into:

- usable derivation source,
- useful ansatz,
- historical fit / calibration,
- computational prototype,
- debt candidate.

Primary targets:

1. Debt 9: scaling from one Standard Model mass anchor.
2. Quark mixing / CKM material from old documents.
3. Fermion table / representation-related evidence.
4. Detection of old multi-anchor fits that must not be mistaken for derivations.

Chat-facing summary should remain descriptive; this repo report may use formulas, tables, and exact source paths.
