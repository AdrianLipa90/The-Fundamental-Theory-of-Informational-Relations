# Hilbert-Kahler Phase Hamiltonian - Formal Derivation Package

This package contains a rebuilt formal derivation of the phase-first Hamiltonian:

Hilbert space -> projective Hilbert/Kahler manifold CP^1 -> Bloch sphere -> Fubini-Study metric -> Berry curvature -> Euler/spin-1/2 constraint -> zeta-polar model axiom -> tetrahedral depth -> Poincare disk -> twin-prime Collatz rhythm -> intention/information Hamiltonian.

## Files

- `main.tex` - LaTeX source.
- `main.pdf` - compiled PDF.
- `scripts/geometry_checks.py` - numerical CP^1/Berry geometry check.
- `scripts/zeta_tetrahedral_map.py` - zeta-polar/tetrahedral depth workbench.
- `scripts/simulate_collatz_poincare.py` - twin-prime Collatz/Poincare/phase closure workbench.
- `figures/*.pdf` - generated figures.
- `results/*.json` - generated simulation outputs.

## Epistemic boundary

Standard layer: CP^1, Bloch sphere, Fubini-Study metric, Berry curvature, spinorial 4pi closure.

Model axioms: zeta-zero pole labels, tetrahedral depth induced from zeta-polar labels, Poincare disk as tetrahedral interference surface, twin-prime Collatz seed as rhythm generator, intention/information phase free Hamiltonian.
