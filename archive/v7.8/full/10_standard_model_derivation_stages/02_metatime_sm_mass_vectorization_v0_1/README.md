# METATIME_SM_MASS_VECTORIZATION_v0_1

Pakiet definiuje warstwę wektoryzacji sektora mas Modelu Standardowego w ramach Metatime / warunku domknięcia Euler--Berry.

Najważniejsze pliki:

- `METATIME_SM_MASS_VECTORIZATION_v0_1.md` — główny dokument.
- `MASS_VECTORIZATION_SCHEMA.json` — jawny schemat wektora \(\mathcal V_f\).
- `sm_representation_vectors.csv` — wektory reprezentacji jednej generacji.
- `mass_action_validation_targets.csv` — cele walidacyjne dla działań masowych, nie dane wejściowe.
- `sm_mass_vectorization_check.py` — kontrola anulowania anomalii.
- `VALIDATION_GATES.md` — zakazy i warunki uznania wyprowadzenia mas.

Status: krok przygotowujący wyprowadzenie \(\mathcal S_f^{EB}\); nie pełny dowód widma mas.
