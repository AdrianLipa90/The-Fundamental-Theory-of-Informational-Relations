# Metatime / Standard Model Derivation — Generation Embedding v0.3

Status: diagnostic bridge, not final canon.

This stage continues the Standard Model derivation after the gauge skeleton and mass vectorization layers.

The central issue is simple: the Standard Model representation vector distinguishes sectors, but it does not by itself distinguish generations. The electron, muon, and tau share the same gauge representation. The up, charm, and top quarks share the same gauge representation. The down, strange, and bottom quarks share the same gauge representation. Therefore a mass hierarchy cannot be derived from the gauge representation vector alone.

The required missing object is a generation vector.

## Canonical interpretation at this stage

The generation vector must be independent of observed masses. It must be computed from structural data only: twin-prime seed, Collatz orbit, Poincare-disk embedding, zeta-polar diagnostic, and later Euler-Berry geometry.

The current layer therefore defines a non-mass-input diagnostic vector. It does not claim to predict masses yet.

## Components

Each candidate generation sector is represented by:

- a twin-prime seed pair;
- a Collatz orbit pair;
- a bounded Poincare-disk embedding diagnostic;
- parity and divergence diagnostics of the two orbit branches;
- a zeta-alignment diagnostic against the first low zeta-zero heights;
- a diagnostic generation-depth score.

The score is not final canon. It is a probe used to test whether the required information exists in the structural layer without using observed mass data.

## Result

The numerical check confirms the structural necessity of this layer. Representation vectors alone cannot carry generation information. Collatz/Poincare/zeta-depth features do carry independent generation-distinguishing data.

This does not yet prove the mass hierarchy. It establishes the next formal bridge needed before mass prediction is even meaningful.

## Consequence for the Standard Model derivation

The derivation must now proceed in this order:

1. Keep the gauge skeleton fixed.
2. Keep anomaly cancellation fixed.
3. Keep standard Yukawa couplings as effective, not fundamental.
4. Introduce the generation vector as an independent structural layer.
5. Build the Euler-Berry action as a function of the full vector, not only the representation vector.
6. Validate against masses only after the action is derived without mass input.

## Validation boundary

This stage uses no observed fermion masses as model input. Observed masses remain validation targets only.

The next unresolved step is to derive the canonical seed triplet and replace the diagnostic generation-depth score by a true Euler-Berry action contribution.
