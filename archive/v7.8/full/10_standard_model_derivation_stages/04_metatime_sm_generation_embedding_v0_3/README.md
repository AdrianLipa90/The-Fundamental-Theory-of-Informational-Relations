# METATIME_SM_GENERATION_EMBEDDING_v0_3

This package continues the Standard Model derivation by introducing a non-mass-input generation vector.

It contains:

- `METATIME_SM_GENERATION_EMBEDDING_v0_3.md` — main report.
- `GENERATION_VECTOR_SCHEMA_v0_3.json` — schema.
- `VALIDATION_STATUS_v0_3.md` — validation boundary.
- `scripts/generation_embedding.py` — reproducible diagnostic script.
- `results/collatz_poincare_generation_candidates.csv` — candidate seed table.
- `results/generation_embedding_summary.json` — machine-readable summary.
- `results/generation_embedding_report.txt` — short plain-text report.

No nested archives. No observed masses are used as model input.
