# METATIME_SM_TETRAHEDRAL_POINCARE_DEPTH_v0_8

This package corrects the Standard Model derivation sequence by placing tetrahedral depth before the Poincare disk.  The Poincare disk is derived from the tetrahedral frame inside the Bloch ball using a projective/conformal Bloch/Klein/Poincare mapping.

The Collatz route is then evaluated on the derived disk.

Observed fermion masses are not used.

Main files:

- `METATIME_SM_TETRAHEDRAL_POINCARE_DEPTH_v0_8.md`
- `TETRAHEDRAL_POINCARE_SCHEMA_v0_8.json`
- `VALIDATION_STATUS_v0_8.md`
- `scripts/tetrahedral_poincare_depth_v0_8.py`
- `results/tetra_poincare_depth_table_v0_8.csv`
- `results/tetra_poincare_depth_summary_v0_8.json`
- `results/tetra_poincare_minimal_triplet_paths_v0_8.svg`
