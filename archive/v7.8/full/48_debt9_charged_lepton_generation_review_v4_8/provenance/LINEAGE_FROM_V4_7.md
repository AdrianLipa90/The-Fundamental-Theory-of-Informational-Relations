# Lineage from v4.7 to v4.8

v4.7 established that H/Htilde + White-Thread improved the quark sector split but left the charged-lepton generation problem unresolved: the muon remained far too high and the tau too low.

v4.8 therefore performs a narrow diagnostic review of the charged-lepton generation layer using only already present project material:

- v4.7 charged-lepton baseline rows;
- v2.9 frozen tau_v2 structural information eigenvalue table;
- document-declared charged-lepton holonomy coefficients Omega_ij and F_ij from the Metatime appendix.

The module does not promote any numeric mass closure. The best candidate is ranked after benchmark comparison, so it is evidence for the next freeze candidate, not itself a prediction.
