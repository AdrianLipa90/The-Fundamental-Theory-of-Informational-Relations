# Collatz Transition Axis and Zeta Imaginary-Zero Axis v1.4

This module corrects the internal order of the Standard Model derivation branch.
The Collatz terminal structure is extended as `4 -> 2 -> 1 -> 1/2`, where `1/2`
is the spin/projective endpoint.  Zeta is placed as an imaginary-zero axis over
the same half-anchor, not as a raw additive mass cost.
