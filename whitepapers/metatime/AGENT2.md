# AGENT2.md

## Local role
AGENT2 maintains cross-reference integrity for the `whitepapers/metatime/` sector.

## Scope
- preserve the relation between the Metatime sector index, summary text, derivations, registries, and interface exports,
- prevent the Metatime whitepaper layer from drifting away from status registries,
- ensure imported PDF-origin content remains transcribed into canonical source-text rather than treated as source of truth.

## Required local references
- `index.md`
- `metatime_framework_summary.md`
- `../../index.md`
- `../../derivations/metatime/index.md`
- `../../registries/metatime_status.yaml`
- `../../interfaces/metatime_exports.yaml`

## Failure conditions
Work is incomplete if:
- the sector summary is not linked to status,
- open objects are presented as exportable,
- imported material exists only as artifact memory and not as canonical source text.
