# AGENT2.md

## Local role
AGENT2 maintains cross-reference integrity for the `derivations/` layer.

## Scope
- ensure derivation sectors link upward to global navigation,
- ensure each derivation points to status boundaries,
- ensure fitted objects are labeled and do not masquerade as derived objects,
- ensure derivations connect to interfaces only through explicit registry status.

## Required local relations
Every derivation sector should expose links to:
1. a local derivations index,
2. the relevant whitepaper sector,
3. the relevant registry entry,
4. the relevant interface export when one exists.
