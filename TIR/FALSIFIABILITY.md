# Falsifiability Criteria

Per Milligan Stage 2 Review: a framework is only scientific if it specifies what would falsify it.

| Criterion | Metatime value | Experimental bound | Status |

| **M_W** (W boson mass) | 77.08 GeV | 80.38 ± 0.01 GeV | **TENSION** — 4% systematic |

## Strong CP: axion vs geometric phase

Metatime predicts θ_QCD = 7.77×10⁻¹⁰, which is:
- **Falsified** by current nEDM if taken at face value (10× above bound)
- **Viable** if a cancellation mechanism reduces d_n by ~10× (not yet proposed)
- **Distinct from axion**: predicts no axion, no axion halo, no ADMX signal

Next-gen nEDM (10⁻²⁸ e·cm sensitivity, 2028+) will test to 0.1× current bound.

## Framework-level falsification

The Metatime framework as a whole would be falsified if:
1. **Any structural choice is found to be mathematically impossible** (e.g., CP³ cannot support the required holonomy)
2. **SM parameters drift** more than 2σ from Metatime predictions in a global fit
3. **A simpler rival framework** (fewer structural choices, same accuracy) is formulated
4. **No cancellation mechanism** for nEDM is found within the geometric phase framework
